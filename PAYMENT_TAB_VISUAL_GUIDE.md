## Color Scheme

- **Yape**: Purple theme (`bg-purple-100`, `text-purple-600`, `border-purple-500`, `peer-checked:bg-purple-600`)
- **Plin**: Blue theme (`bg-blue-100`, `text-blue-600`, `border-blue-500`, `peer-checked:bg-blue-600`)
- **General**: Carbon (dark gray) for buttons and primary elements
- **Borders**: Gray-200 for containers
- **Backgrounds**: White for sections, Gray-50 for page

## Responsive Behavior

- **Desktop**: Full width forms with side-by-side QR preview and upload button
- **Mobile**: Stacked layout for QR preview and upload button
- **Grid System**: Uses existing `space-y-8` and `space-y-4` spacing patterns
- **Image Sizes**: 128px Ã— 128px for QR previews

## Validation Rules

| Field | Validation |
|-------|-----------|
| QR Image | Type: JPG/PNG only |
| QR Image | Size: Max 5MB |
| Phone Number | Type: tel input, no server validation yet |
| Instructions | Optional, multi-line text |
